# Homebrew Tap for Qbit

This is the official Homebrew tap for [Qbit](https://github.com/qbit-ai/qbit), an AI-powered terminal emulator.

## Installation

```bash
brew tap qbit-ai/tap
brew install --cask qbit
```

## Updating

```bash
brew update
brew upgrade --cask qbit
```

## Uninstallation

```bash
brew uninstall --cask qbit
brew untap qbit-ai/tap  # optional
```
